Un sistem de tabele pentru gestiune de informație cu un notepad care salvează textul dinamic.  

Pentru testat în forma livrată:
Asigură-te că ești in folderul "BuLink & Status"
Creează V.E. cu: python3 -m venv venv
Intră în folderul "app"
Instalează dependențele cu comanda: pip install -r requirements.txt
Accesează folderul "BuLink & Status"
Crează user cu comanda: python3 create_user.py
Ca să pornești aplicația scrie în terminal: python3 run.py 
Ține apăsat pe "cmd" (Mac) și dă click pe http://127.0.0.1:5000
Execută comanda "control" + "c" ca să închizi aplicația
